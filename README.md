# DataPulse

DataPulse é uma plataforma de dados corporativos focada em monitoramento de indicadores e insights.
O objetivo é demonstrar um fluxo ponta a ponta: ingestão e tratamento de dados, armazenamento, análises e dashboard.

## Objetivos do projeto
- Construir um pipeline simples de dados (raw → processed)
- Persistir dados em banco (SQLite)
- Calcular KPIs e gerar insights
- Exibir resultados em um dashboard (Streamlit)

## Stack
- Python
- Pandas
- SQLite
- Streamlit

## Estrutura do repositório
- `data/raw/`: dados brutos
- `data/processed/`: dados tratados
- `src/`: scripts do pipeline e análises
- `dashboard/`: app Streamlit
- `docs/`: documentação e imagens
